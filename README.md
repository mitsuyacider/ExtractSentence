## 単語抽出のルール
### ルール
* 「本文」の文章を形態素解析。
* 名詞のみ抽出し、出現回数を記録する。
* 以下の品詞と活用形は除外

```python:除外フィルター
["名詞,代名詞",
"名詞,非自立",
```
